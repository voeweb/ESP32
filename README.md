<h1>ESP32 - Tutorials</h1>

Diese Seite ist speziell für ESP32-Projekte gedacht. Hier finden Sie zahlreiche Tutorials und Programmcodes (C-Codes) für das Arbeiten mit dem ESP32. Ich persönlich bin ein Freund des ESP32 
und es wäre mir eine Freunde meine Begeisterung mit dir zu teilen.
<br>
<br>
Der ESP32 kann entweder seperat als Chip oder aber auch in Kombination mit einem sogenannten "Device-Kit" verwendet werden. Allgemein ist die Verwendung eines solchen "Device-Kits" durchaus zu empfehlen.
Die "Device-Kits" verfügen im Normalfall bereits über die benötigte Peripherie. Solche Kits können beispielsweise mit einem Bluetooth&copy;- oder WLAN-Adapter ausgestattet werden und man kann so mit dem ESP32
kabellos kommunizieren. 
<br>
<br>

du <a href="Fahrzeugsteuerung_mit_dem_Smartphone.md">hier</a>.
